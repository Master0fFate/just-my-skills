### Planner Omega

Path: `skills/planner-omega/SKILL.md`

Builds grounded, verification-first, execution-ready plans for substantial goals. It turns broad build, launch, migration, research, roadmap, and strategy requests into concrete phases, acceptance checks, risks, and immediate next actions.

Use Planner Omega when you want an agent to:

- Preserve the user's goal, scope, assumptions, and decision points.
- Ground plans in current evidence when freshness matters.
- Sequence work by dependency, risk, reversibility, and time-to-value.
- Attach verification gates to each major phase.
- Keep plan changes explicit instead of drifting mid-execution.
